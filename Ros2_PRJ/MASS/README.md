# ROS2 + raspi 를 이용한 자율운항 선박 제작
--

### 구성

    #### 시스템 정보 (예상)
    1. OS : Ubuntu 24.04.2 LTS (noble)
    2. H/W
        1. PC : Raspberry-pi 5 16GB
        2. IMU : SBG Ellipse-E
    3. Libray : ROS2-jazzy, ...

### 개요
    - 연구과제 참여 기관에서 자율 운항 프로그램을 제작 중 이었고 그런 프로그램과 하드웨어를 만들어보고자 하는 마음에 시작하였다. 최대한 간단하게 제작할 예정이며 기간은 2025년 내외로 계획 중이다.
